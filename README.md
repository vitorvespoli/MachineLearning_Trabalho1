# Previsão de Falhas Estruturais em Motor Brushless

<img width="1920" height="1080" alt="capa" src="pics/capa.png" />


Projeto desenvolvido como parte da disciplina de **Machine Learning**, ministrada pelo professor **Samir Angelo Milani Martins**, com foco em análise de dados e aprendizado de máquina para identificação de padrões e possíveis falhas estruturais em motores brushless utilizados em aeronaves de aerodesign.

---

## 🎓 Contexto Acadêmico

Este trabalho foi desenvolvido como atividade da disciplina de **Machine Learning**.

O objetivo da proposta foi aplicar técnicas reais de ciência de dados e aprendizado de máquina em um problema de engenharia, utilizando dados experimentais de um sistema moto-propulsor aeronáutico.

---

## 📌 Sobre o Projeto

Durante a participação da equipe **Trem Ki Voa Micro** na competição **SAE Brasil AeroDesign**, uma falha no motor impediu o voo da aeronave. A partir desse cenário, surgiu a proposta deste estudo:

> Utilizar técnicas de análise de dados para investigar o comportamento operacional do conjunto moto-propulsor e identificar sinais associados a falhas estruturais.

O estudo foi realizado com dados reais coletados em bancada experimental, sem aquisição de novos componentes, utilizando materiais já disponíveis na equipe.

---

## ⚙️ Sistema Avaliado

* **Motor:** Scorpion SII-4035-450
* **Hélice:** Master Airscrew 16×10"
* **Aplicação:** Aeronave SAE AeroDesign – Classe Micro

* Avaliação da consistência dos ensaios realizados
